# 🧠 StudyMind AI — The Elite Learning Copilot

StudyMind AI is a premium, high-retention learning platform designed to turn information into mastery. Built with **Indigo Spatial Minimalism**, it combines the power of Google Gemini AI with a focused, gamified learning loop.

## 🚀 Key Features

### ⚡ Smart Flashcard Pro
Transform any AI output into a structured deck of flashcards with a single click. Our AI identifies key concepts, definitions, and questions to ensure you're studying the most important information.

### 🎯 Interactive Study Mode
Step into Focus Mode for a distraction-free learning experience.
- **Active Recall**: Mark cards as "Known" or "Needs Review."
- **Mastery Scoring**: Get a detailed scorecard at the end of every session.
- **Review Weak Cards**: Smart filtering lets you focus specifically on concepts you haven't mastered yet.

### 🔥 Daily Streak Engine
Stay consistent with our built-in streak system. Track your daily learning habit with a visual badge in your dashboard header that grows as you stay committed.

### 🎨 Spatial UI/UX
Enjoy a state-of-the-art interface featuring:
- **Indigo Spatial Minimalism**: Deep matte palettes and glassmorphic surfaces.
- **Fluid Animations**: Smooth transitions and 3D card-flip interactions.
- **Elite Theme-Switching**: Seamless dark/light mode transition with premium vertical pill-toggle.

## 🛠️ Tech Stack

- **Frontend**: React.js with Vite
- **Styling**: Vanilla CSS (Indigo Design System)
- **AI Engine**: Google Gemini 1.5 Flash
- **Backend/Auth**: Supabase
- **Storage**: LocalStorage Persistence Sync

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/PanthilPatel/StudyMind-AI.git
   cd studymind-ai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory and add:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_key
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```
